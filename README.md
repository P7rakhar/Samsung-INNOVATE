Quickstart

1) Create and activate a Python 3.10+ virtual environment
2) Install dependencies: `pip install -r requirements.txt`
3) Run CLI demo: `python -m src.run_demo`
4) Run dashboard: `streamlit run src/ui/dashboard.py`
